## Ask vscode to tell us what's wrong

- Add this in vscode's config
```json
"[javascript]": {
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode"
},
```
- Install prettier and eslint for vscode.
